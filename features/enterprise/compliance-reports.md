# Compliance Reports

> **Edition: Enterprise** | **Status: Planned**

## Overview

Automated compliance report generation for common regulatory frameworks.

## Planned Capabilities

- PCI-DSS v4.0 evidence reports (Requirements 1, 3, 6, 10, 11)
- HIPAA Security Rule safeguard reports
- SOC 2 Trust Service Categories evidence
- GDPR Article 32 technical measures reports
- Scheduled report generation (daily, weekly, monthly)
- PDF and CSV export

## Current Alternative

Use the audit trail, alert history, and Prometheus metrics to manually assemble compliance evidence. See the [Compliance](../../compliance/pci-dss.md) section for mapping guides.
