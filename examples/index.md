# Examples

Worked scenarios combining multiple eBPFsentinel features for common deployment patterns.

| Scenario | Features Used | Description |
|----------|--------------|-------------|
| [Web Application Protection](scenarios/web-application.md) | Firewall, IDS, L7, Rate Limiting, Alerting | Protect a web application stack |
| [Database Isolation](scenarios/database-isolation.md) | Firewall, IDS, DLP, Audit | Isolate and monitor database servers |
| [PCI Compliance](scenarios/pci-compliance.md) | Firewall, DLP, IDS/IPS, Audit, Threat Intel | Full PCI-DSS deployment |
| [Threat Hunting](scenarios/threat-hunting.md) | Threat Intel, DNS Intelligence, Domain Reputation | Proactive threat detection |

Each scenario includes a complete YAML configuration, CLI commands, and expected behavior.
